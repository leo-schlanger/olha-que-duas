# Olha que Duas

Bem-vindo ao repositório do projeto **Olha que Duas**.

## Sobre
Podcast que dá voz à comunicação, empreendedorismo e novos negócios em Portugal.

## Tecnologias Usadas
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Como Instalar

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Deploy
O projeto pode ser implantado em qualquer plataforma de hospedagem estática (Vercel, Netlify, etc.) rodando `npm run build`.
